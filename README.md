# 💻 WEB_ESTRUCTURA_DE_DATOS

Un simulador interactivo desarrollado en entorno web para la visualización, análisis y modelado de las principales **Estructuras de Datos** lineales y no lineales. Este proyecto busca facilitar el aprendizaje de conceptos abstractos de programación mediante animaciones lógicas en tiempo real.

## 🚀 Tecnologías de Desarrollo
Para lograr una experiencia ágil, ligera y de carga inmediata sin necesidad de dependencias externas, se utilizaron tecnologías web nativas:

*   **HTML5:** Estructuración semántica de la página y maquetación de los paneles interactivos (`index.html`).
*   **CSS3:** Hoja de estilos encargada de la interfaz adaptativa (responsive), colores del entorno y transiciones fluidas de los datos (`styles.css`).
*   **JavaScript (ES6+):** Motor de software del simulador; controla la lógica matemática de las estructuras de datos y la renderización en el DOM (`script.js`).
